## Undo Action

```jsx 
const { showToast, removeToast } = useToast();

showToast({
  message: "Item deleted",
  type: "warning",
  actions: [
    {
      text: "Undo",
      callback: ({ id }) => removeToast(id),
    },
  ],
});
```
Click the button below to trigger a toast:

## Multiple Actions

```jsx
showToast({
  message: "Unsaved changes",
  type: "info",
  actions: [
    { text: "Save" },
    { text: "Discard", className: "text-red-500" },
  ],
});
```
